## 📌 Sobre o projeto

O **My List** é uma aplicação web de gerenciamento de tarefas desenvolvida com **HTML, CSS e JavaScript puro**, permitindo que o usuário adicione, visualize e remova tarefas de forma simples e intuitiva.

O principal objetivo do projeto foi aplicar conceitos fundamentais do desenvolvimento web, incluindo manipulação do DOM, eventos, armazenamento local e construção de interfaces responsivas.

Durante o desenvolvimento, foram explorados:

🔹 Estruturação semântica com HTML<br>
🔹 Estilização e responsividade com CSS<br>
🔹 Manipulação dinâmica do DOM com JavaScript<br>
🔹 Persistência de dados utilizando LocalStorage<br>
🔹 Organização e reutilização de funções

## 🧠 Principais conceitos aplicados

### Manipulação do DOM

Criação dinâmica dos elementos da interface utilizando JavaScript:

🔹 document.createElement()<br>
🔹 appendChild()<br>
🔹 querySelector()<br>
🔹 innerHTML

---

### Eventos

Interação com os elementos da página através de eventos:

🔹 Clique para adicionar tarefas<br>
🔹 Clique para remover tarefas

---

### Estruturas de Dados

Utilização de arrays para armazenar e manipular as tarefas cadastradas:

🔹 push()<br>
🔹 splice()<br>
🔹 map()

---

### LocalStorage

Implementação de persistência de dados no navegador:

🔹 Salvamento automático das tarefas<br>
🔹 Recuperação dos dados ao carregar a página<br>
🔹 Conversão utilizando JSON.stringify() e JSON.parse()

---

### Layout Responsivo

Aplicação de técnicas modernas de CSS:

🔹 Flexbox<br>
🔹 Media Queries<br>
🔹 Hover Effects<br>
🔹 Organização visual adaptável
